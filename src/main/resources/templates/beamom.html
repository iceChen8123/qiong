孕期准备

1. 孕37周开始，洗澡时用小毛巾擦拭手法擦乳头（强度，时间）

2. 乳头的清洁（为什么不用过度清洁；禁止用酒精肥皂清洁；如果处理现在的结痂和蜕皮）

3. 清洁乳房后，用羊脂膏或护理霜涂擦，增加乳头的柔韧性（37周-产后）

4. 乳头凹陷或扁平的妈妈，于36周前往母乳喂养门诊进行早期干预

产后

l 刚开始6个小时如果不吸奶，就把宝宝衣服脱光，留个尿布

l 骨盆带和束腹带都不要用

l 月子不要再吃（多）红糖了,因为有药了.红糖、桂圆、阿娇、人参，都不要吃

l 宝宝要和妈妈睡，不要和月嫂睡，不要把奶吸出来后再喂

l 母乳喂养一定要坚持亲喂和夜间哺乳

l 奶热？奶涨？可以把卷心菜洗干净，放在冷藏备用，然后敷在乳房上。

l 妈妈感冒发烧的时候，能喂奶的。但要戴口罩。如果太严重的话，去看医生。

l 钙要吃到哺乳期结束

科学月子

l 注意休息，适当卧床，避免提重物，以防脏器脱垂

l 适当下地活动，促进胃肠道功能恢复，避免静脉血栓行程

l 每天开窗通风，加强卫生，减少探视

l 注意保暖，避免冷风，切勿捂汗，出汗后及时更换湿衣

l 定期洗澡洗头，注意保暖，及时擦干

l 每天刷牙，餐后及时漱口

l 多喝水，多吃蔬菜水果，保证蛋白、谷物的适量摄入，但不能过量，避免大鱼大肉大量喝油汤 

l 合理用眼，都可以适当少看，不是不能看

l 多与宝宝亲密接触，有利于母婴身心健康

l 避免同房、盆浴

l 产后体力允许，则应适当出门和社交

l 休养环境：

室温在20-26℃左右，如冬天20-24，夏天24-26等。

湿度50—60%为宜。

每天开窗通风2次，每次10-20分钟。

l 洗澡

注意保暖，及时吹干头发。防感冒，洗澡时间不宜过久。

母乳喂养

l 提倡纯母乳喂养至6个月，按需哺乳。

l 每天要做到8-12次以上的吸吮，每次吸吮时间至少15-20分钟。

l 如果吸奶器吸奶，平均3小时1次，每次15分钟。

l 尽量用母乳喂养，不要乱用其他的，哪怕是两岁及以上

l 母乳喂养，宝宝不需要喝水。4-6个月才加水。

l 母乳喂养，不需要排残奶，不需要，不需要。

l 促进泌乳反射：良好感受，自信和孩子在一起；抚摸或看孩子，听孩子哭；背部，肩膀肌肉的放松；沐浴，喝热饮，喝热水，刺激乳头

l 促成功方法：1.母亲自身有信心坚持，2.家庭一致支持，纯母乳，不添加配方奶和水，3.喂养建立成功后，务必做到每日吸吮8-12次，每次一侧乳房吸吮时间至少达到15-20分钟，双侧至少30-40分钟。如果特殊情况做不到，使用手工挤奶或吸奶器。

 

挤奶（吸奶器吸奶）的技巧和频率

l 新生儿吸吮不够，生好就可以挤奶（吸奶），每3小时挤1次，注意夜间也要挤奶（吸奶）。

l 一侧乳房挤3-5分钟换另一侧，反复进行，每次挤奶持续20-30分钟

l 吸奶器挤压10-15分钟即可，中间也要3-5分钟进行更换，并注意开始使用最小档

l 使用情形：新生儿吸吮不够，缓解奶涨，去除乳管堵塞或乳汁淤积，母婴分离。在母亲工作或外出时，在母亲或新生儿生病时，保持泌乳；早产儿、低体重儿，没有吸吮能力时。

新生儿护理

l 一天一次沐浴，天热的话，两次也可以

l 洗澡要用沐浴露，盆里一般滴一两滴就可以了，只要不是泡泡浴，不过洗直接擦干，也是可以的。

l 洗头要用洗发露

l 水温测量，可以用手腕测试

l 棉签使用：

n 婴幼儿棉签（清洁棉签），用于五官护理

n 消毒棉签（酒精），用于脐部护理

l 适合穿连体衣。

l 洗衣机的话，大人小孩分开洗，定期清洁洗衣槽。

l 洗脸：毛巾对折两次，用四个角依次擦左右眼、鼻子和嘴。最后展开用中部干净的，擦额头和脸。

l 护臀膏，每次换尿布都要涂。用清洁棉签涂。 男宝，阴囊和下面的皮肤也要涂，然后左右臀和肛门。女宝，只涂左右臀和肛门，大小阴唇不涂。 肛门都是最后涂。

l 润肤露，洗完澡5分钟内涂。

l 润肤油需要涂皮肤褶皱处和颈部。

l 防晒霜，防晒指数15。先涂宝宝手腕，4-6小时不过敏，再涂。

l 脐带消毒很重要，一天两次，可以一次洗澡时候，另一次随意。一直要到满月。

l 脐部消毒液 碘伏消毒
